# Seamless Student Signup and Onboarding Automation

A secure web-based student onboarding system developed using Flask and MySQL that simplifies the student registration and verification process through OTP authentication, secure password hashing, and role-based dashboards.

---

## Project Overview

This project was developed as part of the iScalePro Pvt. Ltd. internship program.

The system allows students to:
- Register securely
- Verify accounts using OTP
- Login after approval
- Access a personalized dashboard

Admins can:
- Monitor registered students
- Manage onboarding status
- View student details

The application focuses on improving onboarding efficiency, reducing registration errors, and ensuring secure user authentication.

## Features

### Student Features
